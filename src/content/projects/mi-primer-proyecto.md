# Mi Primer Proyecto

Este fue mi primer proyecto completo donde apliqué todos los conceptos que había aprendido.

## Tecnologías usadas

- React como framework de frontend
- TypeScript para type safety
- Node.js con Express para el backend
- PostgreSQL como base de datos

## Lo que aprendí

A través de este proyecto pude aprender:

1. Componentización en React
2. Manejo de estado con hooks
3. Creación de APIs REST
4. Conexión a bases de datos

Fue una experiencia muy enriquecedora que me motivado a seguir creando proyectos.
